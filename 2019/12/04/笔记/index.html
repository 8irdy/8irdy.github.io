实验一：配置一个局域网实现通信。

Router>en 进入特权模式
Router# 特权模式
Router#config t 进入全局模式
Router(config)#hostname R1 给路由器取名叫R1
R1(config)#int f0/0 进入f0/0端口
R1(config-if)#ip add 192.168.10.254 255.255.255.0 配置IP地址
R1(config-if)#no shutdown 开启端口
如何查看配置是否正确？
启动配置文件(startup-config)：配置好以后进行保存的配置文件。
运行配置文件(running-config)：当前已配置但是还未保存或当前运行的配置文件。
R1(config-if)#end 直接退出到特权模式
R1#show startup-config 查看启动配置文件
R1#show running-config 查看运行配置文件
R1#copy running-config startup-config 保存配置文件
R1#write 或 R1#w
R1#reload 重启（重启后running-config的内容会消失，而startup-config的配置会一直存在。
为PC机配置IP地址，子网掩码：255.255.255.0，网关：192.168.10.254
在PC3上测试与路由器和PC4能否通信？
PC>ping 192.168.10.254
Reply from 192.168.10.254: bytes=32 time=1ms TTL=255 表示能够通信

实验二：为提高设备安全，配置console密码保护设备。

R1#config t
R1(config)#line console 0
R1(config-line)#
R1(config-line)#password 123456
R1(config-line)#login
如果需要用户名和密码同时存在，怎么配置呢？
R1#config t
R1(config)#username daiyudong password ciscomima
R1(config)#line console 0
R1(config-line)#login local
设置enable密码：
R1# config t
R1(config)#enable password ciscomima设置明文密码
R1(config)#enable secret ciscomima设置密文密码
当这两个密码同时存在时，密文密码生效。

实验三：为方便管理，配置telnet远程管理设备。

R1(config)#line vty 0 4
R1(config-line)#password ciscomima
R1(config-line)#login
如何远程登录呢？
在PC端的命令提示符中输入：PC>telnet 192.168.10.254
在远程登录时需要用户名和密码呢？
R1(config)#line vty 0 4
R1(config-line)#login local
如何配置管理交换机设备呢？默认情况下，所有端口都在vlan1这个网络，
我们只需要给vlan1这个接口配置IP地址，并配置远程登录密码，就可以
进行远程登录访问了？
Switch(config)#int vlan 1
Switch(config-if)#ip add 192.168.10.253 255.255.255.0
Switch(config-if)#no shutdown
Switch(config)#line vty 0 4
Switch(config-line)#login local
Switch(config)#hostname SW1
SW1(config)#username daiyudong password ciscomima
SW1(config)#enable secret ciscomima

实验四：为提高远程管理设备安全，配置ssh远程管理设备。

Switch(config)#hostname SW1
SW1(config)#ip domain-name daiyudong.com 配置域名（不建议做域名）
SW1(config)#crypto key generate rsa rsa加密
How many bits in the modulus [512]: 1024
SW1(config)#username admin privilege 0 secret ccie
配置仅允许ssh登录
erver(config)#line vty 0 4
Server(config-line)#exec-timeout 10 0 设置超时 分 秒
Server(config-line)#logging synchronous 开启日志同步
Server(config-line)#login local
Server(config-line)#transport input ssh
