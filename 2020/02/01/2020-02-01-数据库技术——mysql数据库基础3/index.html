DQL：select
select语句组成：
select子句
from子句
where子句
order by子句（排序）
limit子句
group by子句（having子句）
join技术

1.检索表中所有数据信息

select * from 表名; 表示表中所有字段

2.检索表中部分字段信息

select 字段1，字段2，…… from 表名;

3.列重命名

select cName as ‘课程名称’ from Courses;
select cName ‘课程名称’ from Courses;
select 字段1 [as] ‘别名’ from 表名;

4.排序（order by)

select * from elogs order by elScore;
select * from elogs order by elScore desc;
select 字段列表 from 表名 order by 字段1 [asc] |desc,[，字段2 [asc] | desc,……];
说明：
A、排序方式包含asc（升序，有小到大，默认值）、desc（降序，由大到小）
B、可根据多个字段排序，但是每个字段有其自身排序方式
C、order by子句出现在where子句后面（如果有where子句）
D、聚合函数可以作为排序依据，如order by count（字段） desc
select * from elogs order by elScore desc;
select * from Teachers order by tAge desc,tNo; 优先考虑tAge，如果有重复则按照tNo排序

select * from Courses order by cName; 因为编码问题可能出现不是我们想要的结果
select * from Courses order by convert(cName using gbk); 将编码改为gbk

5.取多少条数据（limit）

selet 字段列表 from 表名 limit m，n;
注意：
1.limit 关键字后可以只有一个数字，表示取前n条数据
2.limit关键字后有两个数，表示从m+1行开始读取n条数据
查询学生信息表中前五条数据
selet * from Students limit 5;
selet * from Students limit 0,5;
selet * from Students limit 5,5; 从5（不包含5）开始取下一条，总共取5条
任务：检索年龄最小的五位男老师信息
select * from Teachers where tGender=’男’ order by tAge limit 5；

查找年龄最大的老师——年龄最大的不止一个老师的时候
思路：
第一步：找到教师信息表中最大的年龄值
第二步：在教室信息表中找到与年龄值最大值年龄的老师信息
select * from Teachers where tAge=(select tAge from Teachers order by tAge desc limit 1);
select * from Teachers where tAge=(select max(tAge) from Teachers);
重点提醒：聚合函数不能直接出现在where子句中！！

6.聚合函数

max（字段名）：统计返回指定字段的最大值
min（字段名）：统计返回指定字段的最小值
avg（字段名）：统计返回指定字段的平均值
sum（字段名）：统计返回指定字段的和值
count（字段名）：统计返回结果集的信息条数
* count（）
* count（1）
* count（主键字段）
* count（非主键字段）
注意：
1.max（字段）、min（字段）、avg（字段）、sum（字段）、count（非主键字段）在统计时，排除字段值中的null值
2.count（*）、count（1）、count（主键字段）返回表中所有符合条件的信息数目
3.avg（字段）、sum（字段）在统计时针对字符串返回值为0
统计选修记录信息表中，记录信息的数量及考试成绩的平均分
select count(),count(1),count(elno),count(elscore),avg(elscore) from elogs
select avg(cName),sum(cName),max(cName),min(cName) from elogs;

select elNo,count(*) from elogs;——目前来说没有太多意义

7.取消重复值（distinct）

结果集中重复的数据值只会出现一次
select distinct tAge from Teachers;

查询选修考试成绩中，课程编号为2的前十名同学学号及成绩信息
select sID,elScore from elogs where cID=2 order by elScore desc limit 10;
统计课程编号为00002的考试有多少学生未参考（考试成绩为空表示未参考）
select count() from elogs where cID=’00002’ and elScore is null;
select count()-count(elScore) from elogs where cID=’00002’;
查看课程编号为00002，考试成绩为60，70，80，90，100的学生学号
select sID from elogs where cID=’00002’ and elScore in (60,70,80,90,100);
查询参与了选修的同学学号有哪些？
select distinct sID from elogs;
查询参与了选修的同学姓名？
select sName from students where sNO in (select distinct sID from elogs);
select sName from students where exists(select sid from elogs where sid=sno);
外部数据大于内部数据，使用in
内部数据大于外部数据，使用exists

in子句：select 字段列表 from 表A where 表A.字段 in (select 表B.字段 from 表B)；

exists子句：select 字段列表 from 表A where 表A exists (select 主键字段 from 表B where 表B.字段=表A.字段)；

注意：表A.字段与表B.字段有主外关联为佳

合并结果集union
默认将结果集中的重复值取消，如果不想取消重复值，则在union后面增加all
select tName from Teachers union select cName from Courses;

练习：
1、统计学生信息表(Students)中没有填写手机号码的同学有多少人 select count(*) from students where sMobile is null;
2、查看教师信息表(Teachers) 中年龄不小于25岁的老师姓名及性别。
select tName,tGender from teachers where tAge>=25;
3、在选修记录信息表(eLogs) 中，根据学生选修时间排序，将最近登记的五条记录返回，
select * from elogs order by elDate desc limit 5;
4、在课程信息表(Courses)中，将学分分别为3,4,5的课程名称返回。
select cName from courses where cCredit in (3,4,5);
5、在课程信息表(Courses) 中，删除课程描述为空的课程信息。
delete from courses where cDescription is null;
6、将课程信息表(Courses) 中课程描述字段的长度改为200元
alter table courses modify cDescription varchar(200);

7、在选修记录信息表(eLogs) 中，更新学号为20180101，选修课程编号为00002的考试成绩为80。
update elogs set elScore=80 where sID='20180101' and cID='00002';

8、在教师信息表(Teachers) 中，返回年龄范围在24至35的老师名称及编号。
select tName,tNO from Teachers where tAge>=24 and tAge<=35;
9、在课程信息表(Courses)中，查询课程名称中出现了“语言”的课程信息。
select * from courses where cName like '%语言%';

10、在选修记录信息表(eLogs) 中，统计学生所选课程门数(同编号课程只统计一一次)。
select distinct count(cID) from elogs;
11、统计《计算机网络基础》这门课程有多少学生考试及格(- -门课程同一一个学生只能选择一次)。
select distinct count(*) from elogs where elscore>=60 and cid=(select cno from courses where cname='计算机网络基础');

12、统计《计算机网络基础》这门课程缺考的同学有多少。
select distinct count(*) from elogs where elscore is null and cid=(select cno from courses where cname='计算机网络基础');
13、查询张峰老师所授的每门课程名称.
select cName from courses where tID=(select tNO from teachers where tName='张峰');
14、查询哪门课程选修登记的学生人数最多。
select cName from courses where cNO=(select cID from elogs group by cid order by count(cid) desc limit 1);

15、查看课程描述中包含“计算机”或“电脑”字眼的课程名称及其编号。
select cName,cNo from courses where cDescription like '%计算机%' or cdescription like '%电脑%';

连接技术
内连接：inner join
推荐写法：
select 字段列表 from 表A inner join 表B on 表A的主键字段=表B的外键字段 [where 条件表达式];

select 字段列表 from 表A，表B where 表A的主键字段=表B的外键字段 [其他条件表达式]；

例子：
select * from courses;
select * from elogs;
select * from courses as c inner join elogs ase on c.cno=e.cid;

老张老师所授课程的名称
select cName from courses where tid =（select tno from teachers where tname=’老张’);
select cName from courses inner join teacher on tid=tno where tname=’老张’;

检索姓名为张三的学生选修课程《C语言》的考试成绩
select elscore from elogs inner join courses on cid=cno inner join students on sid=sno where sname=’张三’ and cname=’C语言’;

外连接：outer join
左外链接：left outer join /left join
语法：
select 字段列表 from 表A left join 表B on 表A的主键字段=表B的外键字段 [where 条件表达式];
select * from courses as c left join elogs ase on c.cno=e.cid;

查询从未选修过的学生姓名
select sname from students as s left join elogs as e on s.sno=e.sid where e.elNo is null;

分组技术：
select 字段列表 from 表名 [where 条件表达式] group by 分组字段列表 [having 条件表达式] [order by 字段列表]

统计在选修记录信息表中，已登记同学每人选修课程门数
select count() from elogs group by sid;
select count() from elogs group by sid having count()>=2;
select count() from elogs group by sid having count()>=2 order by count() desc;

任务：where和having的异同

#练习：查询已登记的张姓同学的姓名及其选修课程数目，且选修课程门数必须大于一门才能呈现
select sname,count(distinct cid) from students inner join elogs on sno=sid where sname like ‘张%’ group by sid having count()>1 order by count() desc;

#统计《计算机网络基础》这门课程有多少学生考试及格
select count(*) from elogs inner join courses on cid=cno where elscore>=60 and cname=’计算机网络基础’;

#统计不同性别老师的平均年龄及人数
select tGender,avg(tAge),count(*) from teachers group by tGender

#查询哪些课程已有学生选修
select distinct cname from courses inner join elogs on cno=cid where sid is not null;

#统计选修学分达到十分及以上的同学有哪些
select sname from students inner join elogs on sid=sno inner join courses on cno=cid group by sno having sum(ccredit)>10;

select sName from students where sNo=(select sID from elogs where sum(elScore)>10 group by sNo);

select max(elscore) from elogs inner join students on sid=sno where sname like ‘王%’ and year(elogs.elDate)=2018;

select left(tname,1),count(*) from teachers group by left(tname,1);

select max(elscore)-min(elscore) from elogs where cid=(select cno from courses where cname=’C语言’);

select sname from students inner join elogs on sid=sno inner join courses on cid=cno inner join teachers on tid=tno where tname=’王军’ and elscore>=75 and elscore<85;

sql中where和having的区别
用的地方不一样
where可以用于select、update、delete和insert into values(select * from table where ..)语句中。
having只能用于select语句中
执行的顺序不一样
where的搜索条件是在执行语句进行分组之前应用
having的搜索条件是在分组条件后执行的
即如果where和having一起用时，where会先执行，having后执行
子句有区别
where子句中的条件表达式having都可以跟，而having子句中的有些表达式where不可以跟；having子句可以用集合函数（sum、count、avg、max和min），而where子句不可以。
总结
1.WHERE 子句用来筛选 FROM 子句中指定的操作所产生的行。
2.GROUP BY 子句用来分组 WHERE 子句的输出。
3.HAVING 子句用来从分组的结果中筛选行
