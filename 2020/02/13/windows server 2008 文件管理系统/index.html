windows server 2008 文件管理系统

一、文件系统概念

1、FAT文件系统

FAT16最大分区支持2.147G磁盘分区，FAT32最大支持2T磁盘分区。对于400–500MB以下的卷，FAT系统相对于NTFS来说是一个比较好的选择，当卷容量超过1.024GB时，效率就显得很低。

缺点如下：容易受损害、单用户、非最佳更新策略、没有防止碎片的最佳措施、文件名长度受限。

2、NTFS文件系统

NTFS是windows server 2008使用的高性能文件系统，它支持许多新的文件安全、存储和容错功能 ，而这些功能也正是FAT文件系统所缺少的。

（1）NTFS简介

安全、提供文件加密；最大支持2TB硬盘，磁盘容量增大，效率不会降低；可为文件、文件夹赋予权限；具有恢复能力；速度快；支持活动目录和域；支持磁盘配额；支持稀疏文件。

（2）NTFS权限

①标准NTFS文件权限

读取（read）：读取文件内容、查看文件属性与权限等。

写入（write）：可以修改文件内容、在文件后面添加数据或修改文件属性等。

读取和执行（Read&Execute):读取权限和运行应用程序的权限。

修改（Modify）：读取、写入、读取和执行，删除文件。

完全控制（full control）：除了上述权限外还有更改权限与取得所有权的特殊权限。

②标准NTFS文件夹权限

读取（read）：可以查看文件夹内的文件名与子文件夹名、查看文件夹属性与权限等。

写入（write）：可以在文件夹内新建文件与子文件夹，修改文件夹属性等。

列出文件目录（list folder contents）：读取，遍历（traverse folder)权限即打或关闭文件夹。

读取和执行（Read&Execute)：同上，在继承时不同，列出文件目录只会被文件夹继承，而读取和执行会同时被文件夹与文件继承。

修改（Modify）：除了拥有前面的所有权限外，还可以删除此文件夹。

完全控制（full control）：除了上述权限外还有更改权限与取得所有权的特殊权限。

③用户的有效权限

·NTFS权限是可以被继承的

当对文件夹设置权限后，这个权限默认会被此文件夹之下的子文件夹与文件继承。可以设置继承权限还是不继承权限。

NTFS权限是有累加性的

用户或组 权限
用户A 写入
业务部组 读取
经理组 读取和执行
用户A最后的有效权限为：写入+读取+执行

·拒绝权限的优先级较高，继承来的权限其优先级比直接设置的权限低。

用户或组 权限
用户A 写入
业务部组 读取被拒绝
经理组 修改
用户A最后的有效权限为：读取被拒绝

二、文件系统管理

1、访问网络文件

（1）公用文件夹

Windows server 2008系统只有一个公用文件夹，每一位在本地登录的用户都可以访问这个公用文件夹。在客户端访问DC服务器的公用文件夹的方法是：开始–运行–\192.168.10.1。

（2）共享文件夹的创建与访问

①创建共享文件夹

创建方法：右击文件夹–共享…

创建条件：用户必须是Administrators、ServerOperators、PowerUsers等用户组的成员；

​ 如果文件夹位于NTFS分区中，用户至少要有“读取”权限；

​ “用户账户控制”功能已启用（默认已启用）。

用户身份：读 者–读取权限

​ 参与者–更改权限

​ 共有者–完全权限控制

②访问共享文件夹

通过网上邻居访问共享文件夹

通过映射网络驱动器访问共享文件夹

直接输入共享文件夹地址访问共享文件夹

③共享文件夹的管理

共享文件夹的权限

共享权限：读取、更改、完全控制

与NTFS权限结合时以最严格的权限为准

高级共享

通过高级共享可以修改共享名和共享权限

脱机文件

如果设置了访问的共享文件为可脱机使用，那么在通过网络访问这些文件时，这些文件将会被复制一份到用户计算机的硬盘内，可以脱机使用。可通过高级共享进行设置。

隐藏共享文件夹

如果不希望用户在网上邻居中看到共享文件夹，只要在共享后加一个“$”符号就可以将它隐藏起来。

3、利用NTFS管理数据

（1）文件的压缩

NTFS压缩可以节约磁盘空间。当用户或应用程序要读写压缩文件时，系统会将文件自动解压和压缩，从而会降低性能。在NTFS分区内、分区间复制文件夹或在NTFS分区间移动文件夹时，文件或文件夹将继承目标文件夹的压缩属性。而在同一个NTFS分区内移动（剪切）文件或文件夹时，压缩属性被保留。

（2）加密文件系统

加密文件系统（Encrypting File System,EFS)提供文件加密的功能。文件加密后，只有当初对其加密的用户或被授权的用户能够读取，因此可提高文件的安全性。只有NTFS磁盘内的文件或文件夹支持加密，如果将加密文件移动到非NTFS磁盘内会被解密。文件加密与压缩无法并存。

（3）磁盘配额

可以限制用户使用磁盘空间的大小。右击磁盘—配额—设置配额

（4）卷影副本

共享文件夹的卷影副本提供位于共享资源上的实时文件副本。通过使用共享文件夹的卷影副本，用户可以查看在过去某个时刻存在的共享文件和文件夹。手动创建与计划创建。
